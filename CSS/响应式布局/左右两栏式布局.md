## 左右两栏式布局

描述：左侧宽度固定为200px；右侧宽度自适应；  

html代码如下：
```
<div class="main">
    <div class="main-left"></div>
    <div class="main-right"></right>
</div>
```

1. 绝对定位
```
.main{
    position: relative;
}
.main-left{
    position: absolute;
    left: 0;
    top: 0;
    width: 200px;
}
.main-right{
    padding-left: 200px;
}
```
