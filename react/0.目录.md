## 搭建一个 React 项目
- 从脚手架开始，推荐使用 [create-react-app](https://facebook.github.io/create-react-app/)  
create-react-app是facebook官方推出的脚手架。
- 添加路由，推荐使用[react-router](https://reacttraining.com/react-router/web/guides/quick-start)
- 挑选一个UI框架。
