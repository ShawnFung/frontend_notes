## 搭建本地服务器

### live-server
当我们建立一个全静态的前端项目时，如果需要将整个项目运行起来，可以使用 live-server：
1. 安装 live-server
```
npm install -g live-server 或者 cnpm install live-server -gf
```
2. 在项目根目录，执行```live-server```命令，启动服务

如果使用的是 vscode 的话，可以直接安装 Live Server 插件。
