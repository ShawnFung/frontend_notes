### 闭包定义
闭包的设计是为了要解决函数自由变量的问题(Funarg problem)，闭包与自由变量的定义如下:
闭包本质还是函数
Closure说的是函数和它引用的变量之间的关系，和它返回不返回值，返回什么值没有关系
```
var local = '变量'
function foo(){
    console.log(local)
}
```
