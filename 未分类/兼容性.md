## IE
对于IE浏览器，用最新的渲染模式渲染：
```
<meta http-equiv= "x-ua-compatible" content= "ie=edge">
```

在 header 标签中加入下面这段代码：
```
<!--[if lt IE 9]>
  <script src="http://apps.bdimg.com/libs/html5shiv/3.7/html5shiv.min.js"></script>
 <![endif]-->
```
* lt < 小于  
* gt > 大于  
* lte <= 小于或等于  
* gte >= 大于或等于
