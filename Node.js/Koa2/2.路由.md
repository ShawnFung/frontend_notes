# 路由 koa-router 中间件

## 参考文档
1. [koa-router git地址](https://github.com/alexmingoia/koa-router)

## koa-router 使用
```js
var Router = require('koa-router');
var router = new Router();  // 创建一个 router 对象
// 在 router 上添加路径
router
  .get('/', (ctx, next) => {
    // ...
  })
  .get('user', '/users/:id', (ctx, next) => {
   // ...
  })
  .post('/users', (ctx, next) => {
    // ...
  })
// router.routes() 返回 router 中间件
app.use(router.routes())
```
