# 控制器
一个典型的控制器类定义如下：
``` 
namespace app\index\controller;

class Index 
{
    public function index()
    {
        return 'index';
    }
}
```
控制器类文件的实际位置是
``` 
application\index\controller\Index.php
```
使用该方式定义的控制器类，如果要在控制器里面渲染模板，可以使用
``` 
public function index()
{
    $view = new View();
    return $view->fetch('index');
}
```
或者直接使用view助手函数渲染模板输出，例如：
``` 
public function index()
{
    return view('index');
}
```
